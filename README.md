 Этот бот — полезный и интерактивный помощник для студентов и преподавателей техникума, написанный на Python. Он быстро отвечает на основные вопросы и помогает ориентироваться в учебном процессе.
Функции бота
1. Самая крутая группа
Бот может назвать и рассказать, какая группа считается самой активной или успешной.
2. Расписание
Предоставляет актуальное расписание занятий для разных групп, доступ к сменам и преподавателям.
3. Информация о техникуме
Краткие сведения о техникуме.
4. Помощь.
5. Связь с нами
6. Технологии
